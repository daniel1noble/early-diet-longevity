This is the information file for the code used in the paper "Does nutrition early in life affect longevity? A meta-analysis across experimental studies". 

Any comments or questions, please contact Sinead English: sineadenglish[at]cantab[dot]net

The following files are provided in the data folder:
DataLogHR.csv - meta-analysis data based on ln(HR) calculated from survival curves
DataMeanLongevity.csv — meta-analysis data based on mean longevity 

To run the analysis and plot the graphs provided in the paper and ESM, use the file in the ‘code’ folder: 
early-diet-longevity-meta-analysis.R
